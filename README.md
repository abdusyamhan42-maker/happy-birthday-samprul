# 🎉 Happy Birthday! 🎂

Website ulang tahun interaktif yang keren! 

## ✨ Features
- 🎵 Musik birthday otomatis
- 🕯️ Lilin yang bisa ditiup
- 🎆 Animasi kembang api
- 🎈 Balon mengambang
- 🎊 Konfetti celebration
- ✨ Efek mouse trail

## 🚀 Deploy
1. Fork repo ini
2. Edit `index.html` ganti `happy-birthday-samprul`
3. Deploy ke GitHub Pages
4. Share linknya! 🎉
